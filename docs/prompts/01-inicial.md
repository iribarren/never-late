Vamos a desarrollar una aplicación movil para android que sirva para que las personas con TDA o TDAH puedan gestionar mejor su tiempo y las tareas que tiene que hacer.

El desarrollo de la aplicación tiene que servir como un tutorial para aprender Koitlin y desarrollo de aplicaciones Android. Esto debe quedar fijo en tu configuracion para que se haga asi cada vez que se añada una nueva funcionalidad. Las funcionalidades se iran añadiendo de manera progresiva y deben ir incluyendo nuevos conceptos con los que aprender, empezando por lo básico.

Estas son las primeras tareas o funcionalidades a desarrollar, puedes decidir que orden crees que es mejor para que el aprendizaje sea progresivo.

- Preparacion de entorno de desarrollo local. Esto incluye modificar el fichero CLAUDE.md del proyecto para reflejar la estructura del proyecto, como se puede ejecutar en local etc (comprobar los placeholders a sustituir en el fichero)
- Pantalla de inicio para que el usuario introduzca sus datos la primera vez que abre la aplicacion
- Pantalla de inicio cuando el usuario ya ha introducido sus datos, con las distintas opciones de la aplicacion
- Pantalla en la que el usuario puede añadir un contador de tiempo para una tarea
- Pantalla con listado de articulos sobre tecnicas para gestionar el tiempo
- Widget que muestre las tareas pendientes con el tiempo que queda para terminarlas
- Opcion de que las tareas se puedan consultar en la pantalla de bloqueo

Preparar el entorno de desarrollo y adaptar el fichero CLAUDE.md debe ser la primera tarea obligatoriamente.El resto puedes valorar en que orden se deben hacer, o si incluso se puede añadir alguna otra funcionalidad de cara a hacer el tutorial. Las demás tareas puedes planificarse para después
